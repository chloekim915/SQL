CSS 선택자란?

특정한 HTML 태그를 선택하여 스타일을 부여할때 사용되는 식별자(태그, #ID, .Class, 부모/후손, 이벤트 등)

1. 모든 선택자

- HTML 문서 안에 모든 태그를 선택하는 기능


2. 태그 선택자

- 특정 태그를 모두 CSS로 적용하는 방법

    ul 태그의 전체가 변경됨
  • 다른 태그는 변경되지 않음!

    • 3. 아이디 선택자★★★

    - HTML 태그의 속성값에 ID를 지정하여 적용하는 방법

    - id와 style이 1:1 매핑되는 방법으로 많이 사용됨

    id1 영역입니다.
    id2 영역입니다.

    4. 클래스 선택자★★★★★

    - HTML 태그의 속성을 class(그룹)로 지정하여 스타일 적용하는 방법

    - calss와 style 1:N 매핑되는 방법이고, 1:1 매핑 가능. css 적용시 가장 많이 활용되는 기법

    class1 영역입니다.
    class2 영역입니다.
    class3 영역입니다.
    두번째 class1 영역입니다.
    두번째 class2 영역입니다.
    두번째 class3 영역입니다.

    부모영역입니다.
    자식1의 영역입니다. classC 영역입니다.
    자식2의 영역입니다.
    classC 영역입니다. 이곳은 안먹힐 예정
