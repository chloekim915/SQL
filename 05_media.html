멀티미디어 관련 태그

이미지 태그 (img)

- 웹페이지에서 사진이나 이미지를 삽입할때 사용하는 태그

- 링크를 통해 이미지파일을 별도로 가져와서 표시하는 기능


이미지 첨부 1 : 다른 사이트의 이미지 가져오기


이미지 첨부 2 : 내부 파일을 가져오는 법 ※html의 상대 경로로 지정되여야함

상대 경로로 가져오는 법


절대 경로로 가져오는 법 -> 절대 쓰지 말것!


이미지 첨부 3 : alt를 통해 이미지가 안보일때 설명 써보기



이미지 첨부 4 : 이미지 크기 지정




map 태그

- 이미지 구역을 분리하여 링크를 걸어주는 태그

map : 이미지 구역을 생성하는 태그

area : map 내부에서 이미지 구역을 지정해주는 태그


오디오 태그 (audio)

- 웹 브라우저에서 플러그인의 도움 없이 음악을 재생시키는 태그


비디오 태그(video)

- 웹 브라우저에서 플러그인 없이 미디어를 재생시키는 태그

동영상 재생 : 컨트롤 표시 (controls)

동영상 재생 : 재생 전 출력할 이미지 지정(poster), 크기지정(width/height)
