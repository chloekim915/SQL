표(Table)관련 태그

Table 태그

웹 문서에서 Table 형태로 자료를 정리하기 위해 사용되는 태그

행과 열로 구성되어 있고, 만나는 지점을 셀이라 한다.

table 태그 : 테이블을 지칭하는 태그로 가장 밖에 선언된다.

tr : 한개의 행(row)을 나타내는 태그

td : 한개의 열을 나타는 태그

테이블 예시

홍길동 23살
김길동 31살

border 속성 : 테이블의 테두리의 굵기를 조절해주는 속성

최길동 23살 010-1234-4567
김길동 42살 010-4312-4567

caption 태그 : 테이블의 제목을 지정하는 태그

회원 정보
최길동 23살 010-1234-4567
김길동 42살 010-4312-4567

th 태그 : 테이블의 제목 셀을 지정하는 태그

회원 정보
이름 나이 전화번호
최길동 23살 010-1234-4567
김길동 42살 010-4312-4567

colspan : 열 병합 속성

안녕 하이
안녕하세요?

rowspan : 행 병합 속성

안녕 하이1
안녕 하이2
안녕 하이3
안녕 하이1
하이2
하이3

align 속성 : 테이블 전체를 정렬해주는 속성

최길동 23살 010-1234-4567
김길동 42살 010-4312-4567
최길동 23살 010-1234-4567
김길동 42살 010-4312-4567




style : css 속성(스타일)을 걸수 있는 속성

width : 너비

heigth : 높이

text-align : 텍스트 내용의 정렬

border : 테두리 굵기
