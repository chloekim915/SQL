폼 관련 태그

Form 태그

- 사용자가 입력한 값에 대한 처리 방식과 실제 처리해야할 서버의 url을 명시하는 태그

- action : 실제 서버가 처리해야할 url/주소 정보를 기입

- method : get/post를 통해 서버로 전달할 방식을 기입 (default : get)

- submit : form 입력된 데이터를 서버로 전송하는 기능으로 버튼으로 활용

input요소들이 써지는 곳.



fieldset, legend 태그 (잘 사용되지 않는다..)

fieldset : 폼요소를 그룹으로 묶어주는 태그

legend : 폼요소의 그룹의 명칭을 붙여주는 태그
